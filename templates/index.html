About the project

This repository demonstrates an end-to-end ML workflow: data ingestion, EDA, model training, and a minimal web UI to make predictions.

  • Dataset: notebook/data/stud.csv
  • Goal: Predict a student's math_score
  • Language: Python (scikit-learn, Flask)

Model details

Baseline pipeline includes one-hot encoding for categorical features, scaling for numeric ones, and a RandomForest regressor. The saved artifact is model.pkl.

  • Type: scikit-learn Pipeline
  • Model: RandomForest (baseline)
  • Inputs: gender, race_ethnicity, parental_level_of_education, lunch, test_preparation_course, reading_score, writing_score

Inputs

gender: male / female
race_ethnicity: group A / B / C / D / E
parental_level_of_education: some high school / high school / some college / associate's / bachelor's / master's
lunch: standard / free/reduced
test_preparation_course: none / completed
reading_score & writing_score: numeric (0–100)
